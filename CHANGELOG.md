# Changelog — M2M Billing Reference

All notable changes to this reference are documented in this file.

The project follows a **versioned reference model**.
Changes reflect semantic clarification or boundary adjustment,
not feature development.

---

## v1.0.0 — Initial Reference Release  
**Date:** 2026-01

### Included
- Defined scope boundaries for M2M billing semantics
- Established a four-layer reference model:
  - Event
  - Meter
  - Attribution
  - Settlement
- Documented semantic mappings from usage to settlement
- Anchored definitions to stable, primary technical sources

### Notes
- This release intentionally excludes payment, pricing,
  invoicing, and commercial execution.
- The reference is designed for low change frequency.

---

## Versioning Policy

- Minor revisions clarify terminology or boundaries.
- Major versions indicate structural or semantic changes.
- All changes are documented explicitly.
